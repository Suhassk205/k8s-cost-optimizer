**Grader 1: ColdStartGrader** (Lines 94-150)
```python
def grade(self, trajectory: List[TrajectoryStep]) -> float:
    if not trajectory:
        return 0.0  # Edge case
    total_error = sum(step.observation.http_error_rate for step in trajectory)
    avg_error_rate = total_error / len(trajectory)
    score = 1.0 - avg_error_rate
    return max(0.0, min(1.0, score))  # Hard clamp
```

**Formula**: `score = 1.0 - avg_http_error_rate`
**Range**: [0.0, 1.0] ✅
**Normalization**: Length-invariant (normalized by average)
**Edge Cases**: 
- Empty trajectory → 0.0 ✅
- Float tolerance → uses >= comparisons ✅

    if env_path.exists():
        with open(env_path) as f:
            for line in f:
                key, value = line.split("=", 1)
                if key and value and not os.environ.get(key):
                    os.environ[key] = value
