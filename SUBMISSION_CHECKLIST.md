## ✅ 5. Tasks & Graders (3 Required)

### Task 1: cold_start (Easy)
- **Trace**: traces/trace_v1_coldstart.json ✅
- **Grader**: ColdStartGrader ✅
- **Description**: Scale 0→5 replicas without SLA breach (p99 < 300ms)
- **Formula**: `score = 1.0 - avg_http_error_rate`
- **Range**: [0.0, 1.0] ✅

### Task 2: efficient_squeeze (Medium)
- **Trace**: traces/trace_v1_squeeze.json ✅
- **Grader**: EfficientSqueezeGrader ✅
- **Description**: Maintain cpu_steal_pct < 20% across 24-hour cycle
- **Formula**: `score = 1.0 - (violations / trajectory_length)`
- **Range**: [0.0, 1.0] ✅

### Task 3: entropy_storm (Hard)
- **Trace**: traces/trace_v1_entropy.json ✅
- **Grader**: EntropyStormGrader ✅
- **Description**: Issue REBALANCE_NODE before cpu_steal_pct exceeds 20%
- **Formula**: `score = proactive_actions / total_violations`
- **Range**: [0.0, 1.0] ✅
- **Special**: Lookback window = 5 steps

**Status**: ✅ All 3 graders implemented with:
- Edge case handling (empty trajectory → 0.0)
- Hard score clamp: max(0.0, min(1.0, score))
- No division by zero
- Length-invariant scoring
